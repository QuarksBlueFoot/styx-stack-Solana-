This is a guidance-only template generated by `styx init`.

Styx is designed to be **drop-in** for Kotlin apps using a Solana SDK like Artemis and Solana Mobile Stack.

## Recommended wiring
1) Use your existing `Connection`/RPC client and wallet signing flow.
2) Add Styx Kotlin modules for envelope + messaging helpers:
   - `kotlin/styx-envelope`
   - `kotlin/styx-messaging`
   - `kotlin/styx-game` (optional)

3) Keep decrypted content local-only.
   - Persist inbox metadata and cursor locally.
   - Optional outbox queue for reliability.
