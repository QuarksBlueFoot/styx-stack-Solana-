# Styx Init Template — RN + MWA

This folder is generated by `styx init` to help you wire Styx into a React Native app using
Solana Mobile Wallet Adapter (Saga/Seeker) and iOS wallet-adapter compatible flows.

**Tooling-only:** You own keys and wallet UX. Styx provides privacy tooling primitives.

## Files
- `styx.config.json` — defaults + rails preferences
- `styx.setup.ts` — createStyxContext wiring (drop-in)
- `styx.outbox.worker.ts` — best-effort outbox drain/confirm hooks (call on resume / connectivity regained)
